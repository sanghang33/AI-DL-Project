# AI 딥러닝 Project
주제: 심장질환 위험도 예측 모델 개발

1.프로젝트 개요

본 프로젝트는 건강검진 데이터를 기반으로 심장질환 발생 여부를 예측하는
딥러닝 이진 분류 모델을 개발하고, 이를 웹 애플리케이션 형태로 구현하였다.
나이, 혈압, 콜레스테롤, 최대 심박수 등 기본적인 건강 지표를 입력받아
심장질환 발생 확률(%)과 예측 결과(0/1)를 제공하는 것을 목표로 한다.

2.프로젝트 목표 
- 심장질환 고위험군을 조기에 선별
- 의료 데이터 기반 예측 모델 개발 경험
- 딥러닝 모델을 실제 서비스 형태(Streamlit)로 구현
- 데이터 전처리 → 모델 학습 → 평가 → 배포까지 전체 파이프라인 경험

3.사용 데이터 
- 데이터셋: 심장질환 예측 데이터셋
- 데이터 수: 918건
- 변수 수: 12개
- 종속 변수: HeartDisease (0: 정상, 1: 질환)
주요 입력 변수
Age          : 나이
RestingBP   : 안정 시 혈압
Cholesterol : 콜레스테롤 수치
FastingBS   : 공복 혈당 여부
MaxHR       : 최대 심박수
Oldpeak     : 운동 후 ST 하강 값

4.데이터 전처리
- NaN 결측치는 존재하지 않음
- Cholesterol, RestingBP 컬럼의 0값은 측정 누락으로 판단
  → 중앙값(median)으로 대체
- 범주형 변수: One-Hot Encoding
- 수치형 변수: StandardScaler로 표준화

5.사용 모델 
- Optimizer : Adam
- Loss      : Binary Crossentropy
- Metric    : Accuracy, ROC-AUC

6.모델 성능평가
- ROC-AUC : 약 0.94
- Accuracy: 약 87%
- 질환 클래스 Recall ≈ 0.90
- 실제 심장질환 환자를 놓치지 않는 데 초점을 둔 모델
- 의료 도메인 특성상 미탐(False Negative)을 줄이는 방향

7.프로젝트 개선방향
- 제한된 변수로 인해 임상적 해석에 한계 존재
- 추가적인 의료 지표(심전도, 가족력 등) 반영 필요
- 예측 결과에 대한 설명(Explainability) 강화 필요
- 입력값 변화에 따른 시뮬레이션 기능 추가 가능
